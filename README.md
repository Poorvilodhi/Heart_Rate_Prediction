**Heart Rate Prediction (Time Series ML Project)**

This project focuses on predicting heart rate using time series data.
I built this to understand how machine learning models handle sequential data and how past values influence future predictions.

**Project Overview**
Worked with time-based heart rate data
Analyzed trends and patterns over time
Built a machine learning model to predict future heart rate values
Focused on both understanding the concept and building a working pipeline

**Tech Stack**
Language: Python
Libraries: Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

**Approach**
1. Data Preprocessing
Cleaned the dataset and handled missing values
Converted data into a time-based format
2. Exploratory Data Analysis
Visualized heart rate trends over time
Identified patterns and fluctuations
3. Feature Engineering
Created lag features (previous time steps)
Used past values to help predict future ones
4. Model Building
Trained regression models (e.g., Linear Regression)
Used historical data for prediction
5. Evaluation
Compared predicted vs actual values
Used metrics like MAE and RMSE


**Results**
The model was able to capture general trends in heart rate data and make reasonable predictions.
